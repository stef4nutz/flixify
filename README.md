## How to Install?
1. Clone
```bash
git clone https://github.com/stef4nutz/flixify.git
```
2. Install dependencies
```bash
npm i
```
3. Build the app & run it on your server
```bash
npm run build && npm run start
```

Do not forget to claim your TMBD api key from https://www.themoviedb.org/settings/api.
